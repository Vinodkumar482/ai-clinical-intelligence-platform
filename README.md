
# AI Clinical Intelligence & Patient Monitoring Platform

## Overview
This project is an AI-powered platform that helps doctors and hospital staff monitor patient health, generate clinical summaries, detect early risks, and improve clinical decision-making.

## Features
- AI clinical summarization using Amazon Bedrock
- Patient health monitoring
- Early risk detection
- Intelligent alert system
- Clinical dashboard
- Discharge prediction

## Tech Stack
- AWS Bedrock (GenAI)
- AWS Lambda
- Amazon S3
- Amazon DynamoDB
- Amazon OpenSearch
- FastAPI (backend)
- React (frontend)

## Architecture
Data is uploaded to Amazon S3, processed using AWS Lambda, and analyzed using Amazon Bedrock. Results are stored and displayed via a dashboard.
